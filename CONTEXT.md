# Gemma4 Irodori Chat

日本語の音声会話AIアプリに関する言葉を揃えるための用語集。

## Language

**音声セッション**:
**利用者**が音声で**会話ターン**を開始してから、1つの**発話**の聞き取りが完了するまでの音声入力単位。
_Avoid_: マイク入力、録音処理、音声ループ

**会話ターン**:
**利用者**からの1つの入力と、それに対するAIの1つの応答で構成される会話単位。1つの**会話ターン**は、音声またはテキストのどちらからでも始められる。
_Avoid_: セッション、リクエスト、メッセージ処理

**会話履歴**:
過去の**会話ターン**の積み重なり。**会話履歴**はAIが文脈を保って応答するための材料になる。
_Avoid_: チャットログ、メモリ、コンテキスト

**履歴クリア**:
**会話履歴**を空にして、新しい文脈で会話できる状態にすること。
_Avoid_: リセット、削除、初期化

**キャラクター設定**:
AIがどのような人格・話し方・距離感で応答するかを定める会話上の前提。
_Avoid_: システムプロンプト、ペルソナ、キャラプロンプト

**キャラクター画像**:
**利用者**が会話相手として認識するAIの一枚絵。
_Avoid_: アバター、立ち絵、アイコン

**読み上げ設定**:
AIの応答をどのような声・話し方で**読み上げ**るかを定める会話上の前提。
_Avoid_: TTS Caption、音声プロンプト、ボイス設定

**設定保存**:
**クライアントアプリ**で編集した会話上の前提を、**会話サーバー**が以後の**会話ターン**に使える状態にすること。
_Avoid_: Save、反映、更新

**発話**:
**利用者**がAIに向けて話す、意味のある一まとまりの音声。
_Avoid_: 入力、音声、質問

**発話終端**:
1つの**発話**が終わったと判断される境界。**発話終端**が決まると、音声から始まった**会話ターン**は応答生成へ進む。
_Avoid_: 録音停止、無音検知、区切り

**テキスト入力**:
**利用者**が文字で**会話ターン**を開始する入力。
_Avoid_: チャット、手入力、プロンプト

**読み上げ**:
AIの応答を**利用者**に音声として返すこと。音声から始まった**会話ターン**でも、**テキスト入力**から始まった**会話ターン**でも発生しうる。
_Avoid_: TTS、音声生成、再生

**音声チャンク**:
**音声セッション**中に、**クライアントアプリ**から**会話サーバー**へ継続的に渡される短い音声断片。
_Avoid_: 音声ファイル、録音データ、Blob

**利用者**:
このアプリに話しかけ、AIから音声で返答を受ける人。
_Avoid_: ユーザー、話者、クライアント

**クライアントアプリ**:
利用者が操作するアプリ。1つの**クライアントアプリ**は、1つの**会話サーバー**に接続して**音声セッション**を進める。
_Avoid_: Tauriアプリ、フロントエンド、端末側

**会話サーバー**:
**クライアントアプリ**からの要求を受け、AIとの音声会話を成立させる相手側の存在。
_Avoid_: Pythonサーバー、バックエンド、推論PC

**接続**:
1つの**クライアントアプリ**が**会話サーバー**とやり取りできる状態。複数の**接続**が存在しても、複数の**会話ターン**が同時に進むとは限らない。
_Avoid_: セッション、WebSocket、クライアント

**接続先**:
**クライアントアプリ**が接続する**会話サーバー**の場所。
_Avoid_: URL、エンドポイント、サーバーアドレス

**利用可能状態**:
**クライアントアプリ**が**会話サーバー**に接続でき、AI応答と**読み上げ**に必要な相手側の準備も整っている状態。
_Avoid_: 接続済み、online、health

**表示状態**:
**クライアントアプリ**が**利用者**に現在の会話可否を伝えるための状態。
_Avoid_: UI state、ステータス、接続状態

**会話進行**:
1つの**会話ターン**が現在どの段階にあるかを示す状態。
_Avoid_: 内部状態、処理状態、フェーズ

**キャンセル**:
進行中の**会話ターン**を**利用者**が途中でやめること。
_Avoid_: 停止、中断、リセット

**同時利用**:
複数の**クライアントアプリ**が、同じ**会話サーバー**で同時に**会話ターン**を進めること。
_Avoid_: 複数接続、マルチユーザー、並列処理

**推論PC**:
**会話サーバー**が動作する計算資源を持つPC。複数の**クライアントアプリ**が同じ**推論PC**上の**会話サーバー**に接続できる。
_Avoid_: サーバーPC、ホストPC、別PC
